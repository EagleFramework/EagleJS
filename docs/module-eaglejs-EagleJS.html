Return a collection of matched items or created nodes by HTML string.
Example
// string
$('selector');
$('htmlString'); // Create HTML tag

// DOMItem
$(DOMItem);

// DOMItem[]
$(EagleJS);

// string + string
$('selector', 'selector');

// string + DOMItem
$('selector', DOMItem);

// string + DOMItem[]
$('selector', EagleJS);
Parameters:
Name Type Attributes Default Description
selector string | DOMItem | Array.<DOMItem> <optional>
<nullable>
 null A selector to match.
context string | DOMItem | Array.<DOMItem> <optional>
 document A selector to use as context.
