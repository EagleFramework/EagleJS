EagleJS

EagleJS is a jQuery-Like DOM manipulation class for modern browsers.

Constructor

new EagleJS(selectoropt, contextopt)

Source:
  • eaglejs.module.js, line 12
Version:
  • 0.4.4
Copyright:
  • 2020 Cem Demirkartal
License:
  • MIT
See:
  • GitHub
Return a collection of matched items or created nodes by HTML string.
Example
// string
$( 'selector' );
$( 'htmlString' ); // Create HTML tag

// DOMItem
$( DOMItem );

// DOMItemArray
$( DOMItem[] );
$( EagleJS );

// string + string
$( 'selector', 'selector' );

// string + DOMItem
$( 'selector', DOMItem );

// string + DOMItemArray
$( 'selector', DOMItem[] );
$( 'selector', EagleJS );

// EventListener
$(function () {

});
